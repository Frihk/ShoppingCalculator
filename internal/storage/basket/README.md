# Basket Storage

Use this area for basket persistence.

Functions you will likely implement here:

- `SaveBasket`
  Store a basket for later comparison or history.
- `GetBasketByID`
  Load one basket by identifier.
- `ListBaskets`
  Return basket history if needed.
- `UpdateBasket`
  Persist basket changes.
- `DeleteBasket`
  Remove a basket if your app needs cleanup.

If your first version is stateless, this storage may stay small at first.
