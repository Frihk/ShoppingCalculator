# Recommendation Storage

Use this area if you want to store recommendation history.

Functions you will likely implement here:

- `SaveRecommendation`
  Store one generated recommendation result.
- `GetRecommendationByID`
  Load one recommendation.
- `ListRecommendationsForBasket`
  Return recommendation history for a basket.
- `DeleteRecommendation`
  Remove old recommendation results if needed.

This storage is optional for an early version. You may choose to compute recommendations on demand first.
