# Supermarket Storage

Use this area for supermarket persistence.

Functions you will likely implement here:

- `SaveSupermarket`
  Store a supermarket record.
- `GetSupermarketByName`
  Find a supermarket by name.
- `ListSupermarkets`
  Return known supermarkets.
- `ExistsByName`
  Check whether a supermarket is already known.
- `SaveCustomSupermarket`
  Store a user-added supermarket separately if needed.

This repository supports both fixed supermarket lists and user-created supermarkets.
